# AirBnB Seattle Market Analysis — Tableau Dashboard

## Objective
To analyse Seattle AirBnB market trends including pricing 
patterns, property availability, location-based insights, 
and review score distributions to support data-driven 
decision making for hosts and market analysts.

## Dataset
- Source: AirBnB Seattle Open Dataset
- Sheets: Listings, Reviews, Calendar
- Key variables: Price, Neighbourhood, Property Type, 
  Room Type, Availability, Review Scores, Host Details

## Tools Used
- Tableau Desktop
- MS Excel

## Analysis Covered
- Price distribution across Seattle neighbourhoods
- Property type and room type segmentation
- Availability patterns across 30, 60, 90, 365 days
- Review score analysis across multiple dimensions
- Host listing count and superhost distribution

## Dashboard
The Tableau dashboard contains 5 visualisation sheets 
and 1 interactive dashboard. Open the .twbx file in 
Tableau Desktop or Tableau Public to view.
